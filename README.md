# Project Phase 2 - Cloud Dashboard Development

## Stack
- Backend: Python Azure Function
- Storage: Azure Blob Storage
- Frontend: HTML, CSS, JavaScript, Chart.js
- Hosting: Azure Static Web App

## Team Split
- Person A: Azure Function + data processing
- Person B: Frontend dashboard
- Person C: Azure resources + deployment + integration

## Goals
- Read dataset from Azure Blob Storage
- Process data in Azure Function
- Return JSON to frontend dashboard
- Display at least 3 charts
